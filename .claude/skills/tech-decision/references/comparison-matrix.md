## 4. AI/ML 전략

| 전략 | 구현 난이도 | 비용 | 적합한 경우 |
|:--|:--|:--|:--|
| **단순 API 호출** | 쉬움 | 호출당 과금 | 프롬프트로 충분, 자체 데이터 없음 |
| **RAG** | 보통 | 벡터 DB + API | 자체 문서 기반 답변 필요 |
| **Graph RAG** | 높음 | 그래프 DB + API | 복잡한 관계 데이터, 다단계 추론 |
| **Fine-tuning** | 높음 | 학습 비용 높음 | 특정 형식 강제, 대량 데이터 |
| **온톨로지** | 매우 높음 | 구축 비용 높음 | 전문 도메인 지식 체계 |
| **로컬 LLM** | 보통 | 하드웨어 비용 | 데이터 프라이버시, 오프라인 |
