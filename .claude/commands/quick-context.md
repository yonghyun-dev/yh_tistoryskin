현재 프로젝트 상태를 빠르게 파악한다.

다음 파일들을 순서대로 읽고 핵심만 요약한다:

1. `docs/sprint.md` — 현재 스프린트 목표, 단계, 진행 중인 작업
2. `docs/work-log.md` — 최근 작업 로그 (마지막 5개 항목)
3. `docs/context/decisions.md` — 합의된 설계 결정사항
4. `docs/context/session_handoff.md` — 이전 세션 인수인계 (있으면)

출력 형식:
```
## 현재 상태
- 스프린트: #{번호} — {목표}
- 단계: {현재 단계}
- 마지막 작업: {최근 로그 요약}

## 핵심 결정사항
- {결정 1}
- {결정 2}

## 다음 할 일
- {제안}
```
